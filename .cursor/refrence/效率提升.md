## Lightpanda：是一个专门为AI和自动化设计的无头浏览器（跟踪关注即可）

https://github.com/lightpanda-io/browser

Lightpanda 的定位其实很窄：它是无图形渲染的 headless browser，不是给人直接操作的浏览器。“能在 WSL 跑”不等于“能开个页面让真人登录再给 AI 接管”，兼容性和反爬也还没到通用可用的程度。


## agent-browser： 给 AI agent 用的浏览器操作工具
https://github.com/vercel-labs/agent-browser

```shell

agent-browser install  # Download Chrome from Chrome for Testing (first time only)
```
