# mouse_action_component

## 功能
- 鼠标单击与多次点击
- 鼠标双击
- 鼠标移动到指定坐标
- 鼠标滚动
- 鼠标拖拽到指定坐标
- 查询当前鼠标位置

## 导出函数
- `component.handle.mouse_click`
- `component.handle.mouse_double_click`
- `component.handle.mouse_move`
- `component.handle.mouse_scroll`
- `component.handle.mouse_drag_to`
- `component.handle.get_mouse_position`

## 说明
- 鼠标操作依赖系统图形界面会话，需在可操作桌面环境执行。
- macOS 需要为 Python/终端授予辅助功能或自动化权限。

## 依赖
见同目录 `requirements.txt`。
