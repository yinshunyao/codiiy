usage: cursor-cli [--text [PROMPT]] [--danger [FOLDER_PATH]] [--no-format] [--no-color]
                  [--runner-help] [--install] [--resume [SESSION_ID]] [--file FILE_PATH]
                  [--session [CHAT_ID]] [--last-chat-id [SHELL_PID]]
                  [prompt]

A wrapper for cursor-agent with formatted output support. Default mode is streaming with formatted
output.

Runner Options:
  prompt                Prompt to send to cursor-agent (streaming mode by default)
  --text [PROMPT]       Use text output mode instead of streaming. Equivalent to: --output-format
                        text -p PROMPT
  --danger [FOLDER_PATH]
                        Setup .cursor/cli-config.json with extended permissions. Default:
                        ~/.cursor (user home directory)
  --no-format           Disable formatted output for stream-json mode
  --no-color            Disable colored output
  --runner-help         Show this help message and exit
  --install             Install cursor-agent CLI and setup PATH environment variable
  --resume [SESSION_ID]
                        Resume a previous session. If no session ID provided, resumes the last
                        session from this shell.
  --file FILE_PATH      Read task from a file. The prompt will be: '请尝试完成 {absolute_path} 中的任务'
  --session [CHAT_ID]   Lock shell to a chat_id. Use 'new' to create a new session. Run again
                        without args to unlock.
  --last-chat-id [SHELL_PID]
                        Show the last chat_id for a shell session. If no SHELL_PID provided, shows
                        info for current shell.

All unknown arguments are passed directly to cursor-agent.
